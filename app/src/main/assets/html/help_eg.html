2)How do I subscribe?

To subscribe to M Ring service, press 'Activate' button from login page or purchase any ringtones system will register called M Ring servce for you automatically.

3)Can I use it?

Yes, If you are a M Ring subscriber and you have a sufficient balance in your account.

4)Library

You can create your own personal M Ring library. From this library you can choose any ringtones that suits your mood, then set as default.

5)How do I purchase the ringtones?

You can browse through the APP ringtone rank list or scan the QR code and purchase as many ringtones as you want.

6)What if I forget to set a Dial Tune?

It doesn't matter. The system has set a default ringtone for you. If you forget to set your default ringtone for incoming calls the system will apply the default ringtone for you.
